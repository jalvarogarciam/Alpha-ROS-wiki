Ejemplos de proyectos

Guía paso a paso para ejecutar ejemplos prácticos con Alphabot2

Guía de ejemplos

Para poder descargar los ejemplos, hemos de escribir en la terminal wget https://files.waveshare.com/upload/3/39/AlphaBot2-Demo.zip. Una vez realizado, hemos de descomprimir el archivo .zip con ayuda del comando unzip.

Contenido

Tendremos a mano un menú navegable, puesto que esta sección contiene varios apartados con bastante densidad de código.

  • Prueba de motores
  • Control con joystick de 5 direcciones
  • Control remoto infrarrojos
  • Detección de obstáculos por infrarrojos
  • Medición de distancia por ultrasonido
  • Led RGB
  • Prueba de sensores de seguimiento
  • Seguimiento de línea infrarrojo
  • Servo
  • Software mjpg-streamer para monitorización en tiempo real
  • Control por bottle
  • Control remoto mediante software
  • Control remoto mediante bluetooth de la raspberry

Prueba de motores:

Acceda a la terminal de Linux e introduzca este par de comandos: 

            
cd ~/AlphaBot2/python
sudo python AlphaBot2.py

El robot inteligente avanza cuando se ejecuta el programa, accionando el motor para que gire.

Consideramos que la dirección hacia la que apunta la cámara es hacia adelante. Si el Alphabot2 se mueve en la dirección incorrecta, podemos modificar el cableado del motor o cambiar la configuración de los pines, recomendando la segunda opción. Esto se hará modificando el archivo AlphaBot2.py. Esto lo haremos con el siguiente comando:

            
sudo nano AlphaBot2.py

Si las ruedas izquierdas del robot giran en la dirección incorrecta, deberemos intercambiar los ajustes de ain1 y ain2. Para las ruedas derechas, intercambiaremos los ajustes de bin1 y bin2. 

            
def__init__(self, ain1 = 12, ain2 = 13, ena = 6, bin1 = 20, bin2 = 21, enb = 26):

Control con joystick de 5 direcciones:

Acceda a la terminal de Linux e introduzca los siguientes comandos:

        
cd ~/AlphaBot2/python
sudo python Joystick.py

Al pulsar el joystick, suena el zumbador y el motor gira en la dirección indicada. Si las ruedas giran en la dirección incorrecta, deberá modificar la configuración de los pines correspondientes. Las direcciones pulsadas también se imprimen en el terminal.

Control remoto infrarrojos:

Acceda a la terminal de Linux e introduzca los siguientes comandos:

        
cd ~/AlphaBot2/python
sudo python IRremote.py

Puedes controlar el robot inteligente pulsando las teclas numéricas del mando a distancia por infrarrojos:

  • «2» para avanzar
  • «8» para retroceder
  • «4» para girar a la izquierda
  • «6» para girar a la derecha
  • «5» para detenerse
  • «-» y «+» para ajustar la velocidad
  • «EQ» restablece la velocidad predeterminada

(Nota: La codificación de las teclas puede variar con diferentes mandos a distancia por infrarrojos. Puede modificar el programa.)

Detección de obstáculos por infrarrojos:

Acceda a la terminal de Linux e introduzca los siguientes comandos:

        
cd ~/AlphaBot2/python
sudo python Infrarrojos_Obstacle_Avoidance.py

Si no hay obstáculos delante del robot, el LED verde estará apagado. Si se detectan obstáculos, el LED verde se encenderá. En caso de que el LED permanezca siempre apagado o siempre encendido, puede intentar ajustar los potenciómetros situados en la parte inferior del robot para que funcione correctamente.

El robot avanza en línea recta cuando el sensor de la izquierda no detecta ningún obstáculo, y gira a la derecha cuando detecta un obstáculo.

Medición de distancia por ultrasonido:

Acceda a la terminal de Linux e introduzca los siguientes comandos:

        
cd ~/AlphaBot2/python
sudo python Ultrasónico_Ranging.py

El terminal imprime la distancia medida actualmente.

Acceda a la terminal de Linux e introduzca los siguientes comandos:

        
cd ~/AlphaBot2/python
sudo python Infrarrojos_Obstacle_Avoidance.py

El robot avanza en línea recta cuando no detecta ningún obstáculo y gira a la derecha cuando detecta un obstáculo.

Led RGB:

Descarga rpi_ws281x-master.zip a tu Raspberry Pi y ejecuta el siguiente comando para extraer el archivo.


          cd
          wget https://files.waveshare.com/upload/c/c3/Rpi_ws281x-master.zip

Descomprime rpi_ws281x-master.zip

Instala el software necesario.


sudo apt-get install build-essential python-dev scons swig -y

Compile el programa.


cd ~/rpi_ws281x-master
sudo scons

Ejecuta el siguiente comando para realizar la prueba; podrás ver el LED RGB parpadeando.


sudo ./test

Pulse Ctrl + C para detener el programa

Introduzca los siguientes comandos para instalar la biblioteca python.


cd python
sudo python setup.py install

Ejecuta la biblioteca de Python para realizar pruebas y podrás ver el LED RGB parpadeando.


ejemplos de CD
sudo python lowlevel.py

Pulse Ctrl + C para detener el programa

Introduzca los siguientes comandos:


cd ~/AlphaBot2/python
sudo python ws2812.py

Los cuatro LED RGB de la parte inferior del robot inteligente se iluminarán en rojo, verde, azul y amarillo, respectivamente. Luego se apagarán.

Si se muestra un color incorrecto, busque el archivo /boot/config.txt y añada las siguientes líneas. Reinicie el módulo para que los cambios surtan efecto. El LED RGB utiliza control DMA, que puede ocupar los canales DMA de salida de audio de la Raspberry Pi. Por lo tanto, añadir estas líneas podría provocar que la interfaz de auriculares quede inhabilitada.


hdmi_force_hotplug=1
hdmi_force_edid_audio=1

Prueba de sensores de seguimiento:

Acceda a la terminal de Linux e introduzca los siguientes comandos:


cd ~/AlphaBot2/python
sudo python TRSensors.py

En la terminal se muestran cinco conjuntos de datos, correspondientes a cinco sensores. Cuando el robot inteligente se coloca sobre la placa blanca KT, el valor de los datos estará entre 800 y 900. Al suspenderlo en el aire, el valor oscilará entre 0 y 100. Y al situarse sobre la línea negra, el valor se situará entre 100 y 300.

Seguimiento de línea infrarrojo:

Acceda a la terminal de Linux e introduzca los siguientes comandos:


cd ~/AlphaBot2/python
sudo python Line_Follow.py

Mientras el programa se ejecuta, el robot inteligente entra en modo de calibración, moviéndose primero a la izquierda y luego a la derecha. Una vez finalizada la calibración, puedes balancear el robot de lado a lado y la pantalla mostrará la posición actual de la línea negra y los valores de cinco sensores. Coloca el robot inteligente sobre la línea negra y pulsa el botón central del joystick. Ahora podrás ver cómo el robot se desplaza a lo largo de la línea negra y los LED RGB de su parte inferior muestran diferentes colores.

Servo:

Acceda a la terminal de Linux e introduzca los siguientes comandos:


cd ~/AlphaBot2/python
sudo python PCA9685.py

El servo se moverá hacia la izquierda y hacia la derecha, su rango de rotación puede alcanzar los 180 grados.

Software mjpg-streamer para monitorización en tiempo real:

"MJPG-streamer" toma archivos JPGE de la cámara web y los transmite a través de una red basada en IP a varios tipos de espectadores.

  1. Para la cámara con interfaz CSI, es posible que el sistema no pueda detectar el dispositivo en el directorio /dev/video0. En este caso, debe agregar la línea bcm2835-v4l2 al archivo /etc/modules. 

    
sudo nano /etc/modules

    Añade la línea:bcm2835-v4l2

  2. Tras reiniciar, el sistema cargará el nombre del módulo almacenado en este archivo y encontrará el dispositivo video0 en el directorio /dev. (Tenga en cuenta que la letra l en el comando bcm2835-v4l2 es la letra L minúscula). Si aún no puede encontrar el dispositivo video0 en el directorio /dev, y todas las operaciones y comandos son correctos, debe ejecutar el comando: rpi-update.

El software mjpg-streamer incluido en el programa ya está compilado. Solo necesita ejecutar las siguientes líneas para iniciar mjpg-streamer:


cd ~/AlphaBot2/mjpg-streamer/mjpg-streamer-experimental/
sudo ./start.sh

A continuación, abre el navegador e introduce la dirección IP de la Raspberry Pi y el número de puerto: 8080 en la barra de direcciones. El sitio web mostrará la información sobre el transmisor M-JPEG. Haz clic en la opción «Transmitir» para visualizar en tiempo real la imagen capturada por la cámara.

Control por bottle:

Bottle es un framework web de micro Python basado en WSGI, sencillo pero eficiente. Con Bottle, podemos controlar Alphabot2 mediante la web.

Instalar botella


cd
sudo apt-get install python-bottle -y

Frasco de prueba Ejecuta los siguientes comandos para probar la botella.


cd ~/AlphaBot2/Web-RGB/
sudo python main.py

Abre el navegador en tu smartphone (no disponible en PC) e introduce la dirección IP de la Pi con el subfijo 8002. Por ejemplo: 192.168.6.234:8002 (la dirección puede ser cambiable según tus circunstancias), luego podrás acceder a la web, simplemente toca el panel táctil para controlar el color del LED.

Control Alphabot2 Pulsa Ctrl + C para detener el último programa Introduzca los siguientes comandos para ejecutar el programa de control web.


cd ~/AlphaBot2/Web-Control/
sudo python main.py

Abre el navegador de tu smartphone o PC (te recomendamos usar el navegador Chrome), escribe la dirección IP de la Raspberry Pi con el subfijo 8000. Por ejemplo: 192.168.6.234:8000 Acceda al panel de control web y podrá controlar el motor y la cámara mediante botones virtuales.

Puedes configurar el programa para que se inicie al arrancar el sistema. Abre el archivo /etc/rc.local con tu editor y añade la siguiente línea.


sudo vi /etc/rc.local

Agrega los comandos que siguen a los comentarios. Asegúrate de que la línea `exit 0` sea la última. Guarda los cambios y sal.


cd ~/AlphaBot2/mjpg-streamer/mjpg-streamer-experimental/
sudo ./start.sh &
cd ~/AlphaBot2/Web-Control
sudo python main.py &

Dado que la Raspberry Pi utiliza una IP dinámica que puede cambiar cada vez que se reinicia el dispositivo, puede agregar las siguientes líneas en el directorio /etc/dhcpcd.comf para configurar la IP estática:


interfaz wlan0
dirección_ip_estática = 192.168.6.234/24
enrutadores estáticos = 192.168.6.1

“192.168.6.1” es la dirección IP del enrutador y “192.168.6.234” es la dirección IP que se debe configurar, después de lo cual no se debe eliminar el “/24”.

Control remoto mediante software:

Para implementar el control por vídeo del robot inteligente mediante el software de su teléfono móvil o PC, primero debe modificar la dirección IP en el programa.


cd ~/AlphaBot2/App-Control
sudo python main.py

Abre una nueva terminal para ejecutar el script mjpg-streamer. Ejecute el programa AlphaBot.exe en su PC e ingrese la dirección IP, los números de puerto de mjpg-streamer (8080) y del servidor Socket del robot inteligente (8000).

Haz clic en el botón «Conectar vídeo» para conectarte al vídeo. Cuando se establezca la conexión, el botón se pondrá verde. A continuación, haz clic en el botón «Conectar comandos» para conectarte al servidor del robot inteligente. Si la conexión se realiza correctamente, el botón también se pondrá verde. Ahora puedes controlar el robot inteligente pulsando las teclas de la izquierda y controlar la cámara y la dirección con las teclas de la derecha.

A continuación, escanea el código QR que aparece a continuación con tu teléfono móvil y descarga la aplicación correspondiente.

Introduce la dirección IP de la Raspberry Pi y haz clic en el botón Conectar. Verás las imágenes capturadas por la cámara. A continuación, haz clic en el botón Conectar debajo de la imagen. Cuando el puerto de control se haya conectado al programa, este botón cambiará de verde a azul. Ahora puedes controlar las acciones del robot inteligente y su dirección pulsando las teclas correspondientes. Deslizando el control deslizante, puedes cambiar la velocidad del robot.

Abre el archivo /etc/rc.local con tu editor para añadir los siguientes comandos:


sudo nano /etc/rc.local

Agrega los comandos que siguen a los comentarios. Asegúrate de que la línea `exit 0` sea la última. Guarda los cambios y sal.


cd /home/pi/AlphaBot/mjpg-streamer/mjpg-streamer-experimental/
sudo ./start.sh &
cd /home/pi/AlphaBot/App-Control
sudo python main.py &

Deberás conectar la Raspberry Pi al router cada vez que la reinicies. También puedes habilitar la función de punto de acceso (AP) del adaptador de red inalámbrica de la Raspberry Pi, configurándola como servidor. Para obtener más información sobre su funcionamiento, consulta el tutorial de Raspberry Pi.

Control remoto mediante bluetooth de la raspberry:

1) Actualice los paquetes correspondientes de Bluetooth e instale el software del que depende.


sudo apt-get update
sudo apt-get upgrade -y
sudo apt-get dist-upgrade -y
sudo apt-get install pi-bluetooth bluez bluez-firmware blueman -y

2) Agregue el usuario pi al grupo Bluetooth.


sudo usermod -G Bluetooth -a pi

3) Reinicie la Raspberry Pi.


sudo reboot

4) Inicie/Agregue el SPP y encienda el dispositivo Bluetooth.


sudo nano /etc/systemd/system/dbus-org.bluez.service

Modifique dos líneas en el siguiente archivo:

Después de reiniciar la RPi, ingrese el comando hciconfig (similar al comando ifconfig) para verificar el servicio Bluetooth.

Si el Bluetooth funciona correctamente, verá el dispositivo hci0 en la pantalla. Si el Bluetooth no funciona, no se detectará ningún dispositivo Bluetooth. Nota: Si la línea dtoverlay=pi3-miniuart-bt existe en el archivo /boot/config.txt, debe comentarse.

5) Conéctese al dispositivo Bluetooth Ingrese el siguiente comando para ingresar a la interfaz de control de Bluetooth (puede ingresar el comando help para consultar la lista de comandos después de ingresar a la interfaz).


sudo bluetoothctl

Una vez dentro de la interfaz, introduzca las siguientes líneas (puede utilizar el comando help para consultar la lista de comandos tras acceder a la interfaz).


agente en
agente predeterminado

Tras ejecutar el comando scan on, podrá ver la dirección física del dispositivo Bluetooth escaneado, similar a XX:XX:XX:XX:XX:XX.

Para el emparejamiento Bluetooth, utilice el siguiente comando:


par XX:XX:XX:XX:XX:XX

Cuando el emparejamiento se haya realizado correctamente, agregue el dispositivo a la lista de dispositivos de confianza.


confianza XX:XX:XX:XX:XX:XX

A continuación, conéctese al dispositivo.


Conectar XX:XX:XX:XX:XX:XX

Finalmente, introduce el comando «quit» para salir de la interfaz. Tras realizar los ajustes anteriores, al encender el dispositivo Bluetooth, la Raspberry Pi se conectará automáticamente a él. Nota: Si no puede establecer la conexión mediante la línea de comandos, puede acceder a la interfaz gráfica para realizar la conexión.

Haz clic en el icono de Bluetooth y añade un nuevo dispositivo.

Seleccione el dispositivo Bluetooth correspondiente. Asegúrese de que el Bluetooth de su teléfono móvil esté activado y configurado como visible.

En la RPi, seleccione la opción Aceptar para emparejar. Y al mismo tiempo, en el teléfono móvil, pulse la opción Emparejar.

A continuación, es posible que vea un mensaje que dice: “Emparejamiento realizado correctamente, pero este dispositivo no tiene servicios que se puedan usar con Raspberry Pi”. Haz clic en Aceptar. Ahora, la conexión se ha establecido.

Ejecute los siguientes comandos para iniciar el programa:


cd ~/AlphaBot2/Bluetooth-Control
sudo rfcomm watch hci0 1 python bluetooth.py

A continuación, escanea el código QR que aparece más abajo con tu teléfono móvil y descarga la aplicación correspondiente. (Este es el software para controlar el módulo Bluetooth de modo dual. Si ya tienes instalado este software en tu teléfono móvil Arduino, puedes omitir este paso). Inicie la aplicación y haga clic en la opción Escanear. (Nota: asegúrese de que el Bluetooth de su teléfono móvil esté activado). Normalmente, verá los dispositivos Bluetooth disponibles en la lista. A continuación, seleccione la opción «raspberrypi». Vaya a la siguiente página y seleccione la opción Control remoto.
