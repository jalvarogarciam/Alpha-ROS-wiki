Instalación de ROS y Ubuntu

Para ponernos manos a la obra, primero tenemos que instalar la distribución de Ubuntu y sobre ella, ROS Noetic Ninjemys. Instalación de Ubuntu 20.04.6 LTS La mejor forma para instalar ésta distribución de ubuntu de forma no local, es instalarla como máquina virtual dentro de nuestro equipo. Para ello, podemos seguir los pasos de la guía de instalación de Ubuntu que podemos encontrar en la web oficial de la Oficina de Software Libre de la UGR, usando la ISO de Ubuntu 20.04.6 que podemos descargar en la web oficial de Ubuntu.

Instalación de Ros Noetic Ninjemys.

Paso 1: Configuración de los Repositorios de Ubuntu

El sistema debe estar configurado para aceptar software desde packages.ros.org.

  • 1. Añadir el repositorio de ROS. Ejecuta el siguiente comando en una terminal para añadir el repositorio oficial de ROS a la lista de fuentes de apt. 

    
sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

  • 2. Añadir la clave de seguridad. Esta clave es necesaria para que el gestor de paquetes confíe en el repositorio de ROS. 

    
sudo apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654

Paso 2: Instalación de ROS Noetic

Con los repositorios configurados, proceda con la actualización e instalación. Para ello, primero actualizamos el índice de paquetes con 


sudo apt update

Llegados a este punto, tenemos varias opciones de versiones a instalar:

  • Si tienes buen espacio y conexión, instala la versión Desktop-Full (recomendada). Esta versión incluye el núcleo de ROS, herramientas 2D/3D (RViz), simuladores (Gazebo) y librerías de navegación. 

    
sudo apt install ros-noetic-desktop-full

    (Nota: Esta instalación puede tardar varios minutos dependiendo de la conexión a internet y el rendimiento del sistema.)
  • Si no, opta por la versión ROS-Base, más ligera y rápida de descargar pero más limitada. 

    
sudo apt install ros-noetic-ros-base

ROS cuenta con diversidad de paquetes que podemos observar en la web de ROS Index ó usando el comando apt search ros-noetic. Éstos paquetes pueden ser instalados usando el comando: 


sudo apt install ros-noetic-<NOMBRE_DEL_PAQUETE>

Paso 3: Configuración del Entorno

Es necesario configurar el entorno de la terminal para que reconozca los comandos de ROS. Este comando añadirá automáticamente la configuración de ROS al archivo de inicio de la terminal. 


echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc
source ~/.bashrc

Paso 4: Instalación de Dependencias Esenciales (rosdep)

Rosdep es una herramienta fundamental que instala automáticamente las dependencias de los paquetes de ROS. Para instalarla e inicializarla, ejecuta los siguientes comandos: 


sudo apt install python3-rosdep
sudo rosdep init
rosdep update
