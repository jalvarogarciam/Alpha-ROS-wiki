Activación de RViz

Guía paso a paso para activar y utilizar RViz con ROS

Activación de RViz

Si no tienes Rviz instalado:

$ sudo apt install ros-noetic-rviz

Para ejecutar la visualización:

  • Lanza Gazebo con un mundo.
  • Abre Rviz en otra terminal con el comando:

    $ rviz

  • Configura el "Fixed Frame" a odom.

  • Añade "RobotModel" y "LaserScan" (Topic: /scan) desde el botón Add, (Robot Model nos permite cargar el modelo de robot que vayamos a utilizar y LaserScan es un tipo de Display para mostrar datos provenientes de sensores LIDAR 2D planos, los muestra mediante una nube de puntos).

Si el robot aparece en blanco y RobotModel en rojo (error), ejecuta en una nueva terminal:

$ rosrun robot_state_publisher robot_state_publisher

Guardamos la configuración por ejemplo en el escritorio, File → Save as, y escribimos el nombre que queramos, por ejemplo, mi_simulacion_robot

Si queremos volver a abrir esta configuración tendremos que abrir una nueva terminal y escribir:

$ rviz -d [ruta donde se encuentre el archivo de la configuración]

En caso de que al abrirlo nos salga el robot en blanco abrimos una nueva terminal y ejecutamos:

$ rosrun robot_state_publisher robot_state_publisher
