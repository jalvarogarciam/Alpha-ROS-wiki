Introducción

Kit de construcción de robots AlphaBot2 para Arduino/Pi/PiZero

Los kits de robot AlphaBot2-Pi incluyen un chasis (chasis AlphaBot2-Base) y una placa adaptadora AlphaBot2-Pi. El robot es compatible con Raspberry Pi mediante una placa adaptadora AlphaBot2-Pi.

Gracias a su diseño modular altamente integrado, el montaje es muy sencillo: solo hay que encajarlo, sin soldar ni cablear. Tras unos minutos de montaje, estará casi listo. Nuestros códigos de demostración de código abierto le ayudarán a empezar rápidamente.

Los accesorios pueden variar según el producto. Esta página describe los accesorios compatibles con la placa Raspberry Pi.

  1. Interfaz de control AlphaBot2: para conectar distintos tipos de placas adaptadoras de controlador.
  2. Interfaz del módulo ultrasónico
  3. Indicadores para evitar obstáculos
  4. Rueda omnidireccional
  5. ST188: sensor fotoeléctrico infrarrojo reflectivo, para evitar obstáculos
  6. ITR20001/T: sensor fotoeléctrico infrarrojo reflectivo, para seguimiento de línea
  7. Potenciómetro para ajustar el rango de detección de obstáculos
  8. Controlador de motor de doble puente H TB6612FNG
  9. Comparador de voltaje LM393
  10. Motorreductor micro N20, relación de reducción 1:30, 6 V/600 RPM
  11. Ruedas de goma, diámetro 42 mm, ancho 19 mm
  12. Interruptor de encendido
  13. Portapilas: admite baterías 14500
  14. WS2812B: LED RGB de color verdadero
  15. Indicador de carga de la batería
  16. Puerto de carga de batería USB de 5 V
  17. Indicador de encendido

  1. Interfaz de control AlphaBot2: para conectar la base AlphaBot2.
  2. Interfaz Raspberry Pi: para conectar Raspberry Pi 3 Modelo B.
  3. Interfaz de servo.
  4. USB a UART: fácil de controlar la Raspberry Pi mediante UART.
  5. LM2596: Regulador de voltaje de 5 V.
  6. TLC1543: Chip de adquisición AD de 10 bits, permite que la Raspberry Pi utilice sensores analógicos.
  7. PCA9685: controlador de servo, permite una rotación más suave del cabezal panorámico.
  8. CP2102: Convertidor USB a UART.
  9. Palanca de mando.
  10. Receptor de infrarrojos.
  11. Zumbador.
