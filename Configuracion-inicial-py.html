Configuración inicial

Aprende a configurar tu Alphabot2 para su uso con Raspberry Pi

Configuración del software

Conéctate a la red wifi.

  1. Conecte una pantalla, un teclado y un ratón a la Raspberry Pi y encienda la Raspberry Pi.
  2. Conecta la Raspberry Pi a la red Wi-Fi pulsando el icono de Red en la esquina superior derecha de Raspbian.
  3. Tras conectarte, pulsa las teclas Ctrl + ALT + T para abrir la terminal de Raspbian.
  4. Escribe el comando ifconfig para comprobar la dirección IP de Raspbian.

Acceso remoto a Raspberry Pi

Comenzaremos descargando el software Putty en nuestro PC. Al abrirlo, introduciremos la dirección IP comprobada anteriormente. Una vez tengamos todo a punto, le daremos clic a abrir. Una vez hemos accedido, nos identificaremos en Raspberry Pi con el nombre de usuario predeterminado (pi) y la contraseña (raspberry).

Habilitación de interfaces

1. Escriba en la terminal de raspberry el comando para abrir raspi-config

Tras esto, en el nuevo menú seleccione la opción Interfacing Options.

sudo raspi-config

2. Seleccione en las opciones de interfaz la opción cámara, luego confirme dos veces y acepte.

3. Seleccione en las opciones de interfaz la opción SPI, confirme y acepte la configuración.

4. Seleccione en las opciones de interfaz la opción I2C, confirme y acepte la configuración.

5. Seleccione en las opciones de interfaz la opción UART, primero dele a no, luego confirme y por último acepte la configuración.

6. Para terminar, seccione finalizar para poder salir. Pulse sí si le requiriese reiniciar; y en caso de que la opción no aparezca puede hacerlo manualmente usando el comando sudo reboot en la terminal raspberry.

7. Una vez se reinicia, Putty se desconecta, tan solo acepta y espera a que se reinicie. Una vez listo, haz clic en Putty y seleccione Reiniciar sesión, iniciando de nuevo con las credenciales.

Configuración con RealVNC Viewer

Podremos descargar VNC Viewer en el siguiente enlace: Instalación VNC Viewer

Una vez seleccionado nuestro sistema operativo, lo instalaremos y procederemos a configurarlo para conectarnos a Alphabot. Hemos de ver que este esté conectado a nuestra red y, tras ello, descubrir su IP, pudiendo conectar un mini HDMI, un teclado y un ratón a nuestro robot para tratarlo como un ordenador y facilitar nuestra tarea.

Una vez sepamos los datos, abriremos RealVNC Viewer y seleccionaremos la opción Archivo->nueva conexión. Una vez hecho esto, podremos nombrar como queramos esta conexión e introduciremos la IP previamente descubierta. Una vez nos hayamos conectado tendremos que poner el usuario (pi) y su respectiva contraseña (raspberry).

Configuración con WinSCP

WinSCP (Windows Secure Copy) es un cliente gráfico de código abierto para Windows que permite la transferencia de archivos.

Funciona como un puente visual entre el ordenador (Windows) y el "cerebro" de nuestro robot .En lugar de usar comandos complejos en una terminal para mover archivos, WinSCP ofrece una interfaz visual de ventanas.

Manual de uso: Paso a paso

Para conectar al robot, necesitaremos tres datos clave: su dirección IP, su usuario (pi) y su contraseña (raspberry).

  1. Configuración de la Conexión

    2. Al abrir WinSCP, verás la ventana de inicio de sesión. Configúrala así:

    • Protocolo de archivo: Selecciona SFTP (es el estándar seguro que usa el protocolo SSH).
    • Nombre o IP del servidor (Host name): Escribe la dirección IP de tu robot.
    • Número de puerto: Déjalo en 22 (estándar SSH).
    • Usuario y Contraseña: Ingresa las credenciales de tu robot.
    • Guardar: Púlsalo para no escribir esto datos cada vez que abras WinSCP.
    • Haz clic en Conectar.

    Nota: La primera vez que te conectes, saldrá una advertencia de seguridad sobre una "clave de host" (Host Key). Haz clic en "Sí" para confiar en el robot.

  2. Interfaz (Panel doble)

    3. Una vez conectado, verás la interfaz clásica de "Comandante":

    • Panel Izquierdo: Son los archivos de tu ordenador (Local).
    • Panel Derecho: Son los archivos del robot (Remoto).

  3. Transferencia de archivos.

    4. Será tan simple como arrastrar y soltar.

    • Para subir al robot: Arrastra un archivo desde la izquierda (tu PC) hacia la derecha (el robot).
    • Para descargar del robot: Arrastra desde la derecha hacia la izquierda.

Instalación de bibliotecas

Para poder utilizar correctamente el robot, necesitaremos instalar las siguientes bibliotecas. Accediendo a la terminal, ejecutaremos los siguientes comandos:

            
sudo apt-get update
sudo apt-get install ttf-wqy-zenhei
sudo apt-get install python-pip
sudo pip install RPi.GPIO
sudo pip install spidev
sudo apt-get install python-smbus
sudo apt-get install python-serial
sudo pip install rpi_ws281x
