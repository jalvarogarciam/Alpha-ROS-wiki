Conexión y control remoto

Aprende a conectar y controlar tu Alphabot2 de forma remota

AlphaBot2 para Raspberry Pi Zero – PINOUT

Conoceremos ahora cómo funciona nuestro robot en detalle, conociendo su conexión.

1. Motores de las ruedas:

Los motores de las ruedas se dividen en dos: el motor izquierdo (A) y el motor derecho (B), realizándose el control de velocidad de cada motor mediante señales PWM. El motor izquierdo utiliza el pin ENA (GPIO6) , mientras que el motor derecho usa el pin ENB (GPIO26). Para controlar la dirección de giro, el motor izquierdo emplea los pines AIN1 (GPIO12) y AIN2 (GPIO13), y el motor derecho utiliza BIN1 (GPIO20) y BIN2 (GPIO21).

2.Servo motor:

El servo motor no se conecta directamente a los pines GPIO, sino que se gestiona a través del controlador PCA9685, el cual se comunica con la Raspberry Pi mediante el bus I2C. Este módulo permite controlar la posición del servo de forma precisa sin sobrecargar la CPU principal.

3.Sensores de distancia ultrasónico:

En cuanto a los sensores de distancia ultrasónicos, el pin GPIO22 se utiliza como Trigger (TRIG) para emitir la señal ultrasónica, y el pin GPIO27 como Echo (ECHO) para recibir la señal reflejada y calcular la distancia al obstáculo.

4.Sensores de líneas infrarrojos:

Los sensores de línea infrarrojos (IR) emplean un convertidor analógico-digital (ADC) TLC1543 para transmitir la información a la Raspberry Pi. Este ADC se comunica mediante el bus SPI, utilizando los pines GPIO5 (CS), GPIO25 (CLK), GPIO24 (Address) y GPIO23 (DataOut). Gracias a esta conexión, la Raspberry puede leer los valores analógicos de los sensores de línea y determinar si el robot se encuentra sobre una línea o fuera de ella.

5.Otros componentes:

Además de todos estos componentes fundamentales, el AlphaBot2 incorpora otros componentes de apoyo. El zumbador (buzzer) es uno de ellos, estando controlado mediante el pin GPIO4, pudiendo utilizarse para emitir señales acústicas. También dispone de un LED RGB conectado al GPIO18, que permite generar distintos colores de luz para indicar estados o eventos del sistema.
